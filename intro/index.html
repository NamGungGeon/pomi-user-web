왜 포미를 사용해야 할까요?

  • 귀가 보호모드
  • 여성안심 스카우트 간편예약
  • 지역별 위험정보 확인
  • 근처 안전정보 확인
  • 휴대폰 흔들어 신고

귀가 보호모드

귀갓길을 설정하면 자동으로 사용자의 위험상황을 감지합니다. 유사 시에는 미리 지정한 보호자에게 위험 알림 문자가 발송되어 신속한 조치를 취할 수 있습니다.

여성안심 스카우트 간편예약

스카우트 대원분들이 직접 자택까지 귀가를 도와드리는 여성안심 스카우트 서비스를 손쉽게 예약하세요.

지역별 위험정보 확인

지역 주민들이 직접 등록하는 실시간 위험정보를 확인하세요. 내 귀갓길 주변의 위험을 알림설정하고 지역별 커뮤니티를 통해 더 자세히 알아볼 수 있습니다.

근처 안전정보 확인

귀갓길을 설정하면 자동으로 사용자의 위험상황을 감지합니다. 유사 시에는 미리 지정한 보호자에게 위험 알림 문자가 발송되어 신속한 조치를 취할 수 있습니다.

휴대폰 흔들어 신고

보호모드 시 위급한 상황에서 휴대폰을 흔들면 보호자에게 위험알림이 전송됩니다.

자주 문의하는 질문

안심귀가 스카우트 신청에 문제가 있습니다

현재 서울지역에서만 스카우트 기능을 지원합니다
추후 다른 지역에서도 지원 가능하도록 기능을 추가하겠습니다.

더불어, 여성안심 스카우트는 평일 21시 30분 ~ 24시 30분(월요일은 21시 30분 ~ 23시 00분까지)까지만 운영됩니다.

지정보호자는 왜 필요한가요?

사용자가 위험에 빠졌다고 판단되면, 등록한 보호자에게 사용자의 위치를 지속적으로 전송합니다.
위험 상황에 빠르게 대처할 수 있도록, 반드시 유효한 전화번호를 입력해주세요.

앱 사용에 문제가 있습니다

앱 관련 문의나 오류 및 개선사항은 앱 내에서 신고해주시면 빠른 답변 드리겠습니다.
